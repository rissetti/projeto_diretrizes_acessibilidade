Passo a passo para a instalação do Visual Studio code:

  • Primeiro passo: Acessar o linkhttps://code.visualstudio.com/
  • Segundo passo: Selecione o seu sistema operacional e faça o download;
  • Terceiro passo: Após concluir o download abra o Visual Studio Code;
  • Quarto passo: Aparecerá uma janela com os termos de uso, leia os termos ;
  • Quinto passo: após ler os termos, clique em “aceitar os termos de uso” para prosseguir;
  • Sexto passo: Após, aparecerá a opção Selecionar Tarefas adicionais, selecione todas e clique em próximo;
  • Sétimo passo: Após, clique em instalar e o computador fará a instalação Do Visual Studio Code;

    Configurações de linguagem do Visual Studio Code:

  • Primeiro passo: Abra o Visual Studio Code e Pressione as teclas Ctrl, Shift, X para acessar a opção “EXTENSIONS”;
  • Segundo passo: No campo de pesquisa digite “Português”;
  • Tercerio Passo: Selecione o pacote "Portuguese (Brasil) Language Pack for visual Studio Code";
  • Quarto passo: Abrirá uma aba ao lado com todas as informações do mesmo, Clique em instalar;
  • Quinto passo: Após a instalação, Pressione as teclas Ctrl, Shift, P para abrir a paleta de comandos;
  • Sexto passo: Digite “display lenguage” para filtrar e exibir o comando “configurar o idioma de exibição”;
  • Sétimo passo: Pressione a tecla enter e será exibida uma lista de idiomas instalados por localidade;
  • Oitavo passo: Selecione “Português (Brazil)”, e feche o Visual Studio code;
  • Nono passo: Ao abrir novamente, estará no idioma Português(Brazil).

    Passo a passo para a instalação plug in C/C++:

  • Primeiro passo: Pressione as teclas Ctrl, Shift, X para acessar a opção “EXTENSÕES”;
  • Segundo passo: No campo de pesquisa digite C/C++;
  • Terceiro Passo: Selecione o pacote “C++ Extension Pack”;
  • Quarto passo: Abrirá uma aba ao lado com todas as informações do mesmo, clique em instalar.

    Passo a passo para a instalação plug in JAVA:

  • Primeiro passo: Pressione as teclas Ctrl, Shift, X para acessar a opção “EXTENSÕES”;
  • Segundo passo: No campo de pesquisa digite “JAVA”;
  • Terceiro Passo: Selecione o pacote “EXTENSION PACKS FOR JAVA”;
  • Quarto passo: Abrirá uma aba ao lado com todas as informações do mesmo, clique em instalar.
